# Escape-Room-Challenge

## Overview
A 3D horror escape room game where players must solve puzzles and uncover clues to escape before time runs out.  

## Theme
The project creates a tense horror atmosphere with suspenseful interactions, ambient sounds, and ominous lighting.

## Collaboration
This repository is shared among team members for our capstone project.
Everyone will contribute through branches, commits, pull requests, and merges.

## Tools
- Unreal Engine (game development)
- VS Code 
- GitHub (version control and collaboration)
